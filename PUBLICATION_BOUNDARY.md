# Publication Boundary

This repository is a public verification registry, not a development archive.

Public visibility does not:

- create an offer, security, token, asset, land right, governance right,
  cash-flow right, certification, or warranty;
- turn hypotheses, specifications, runtime output, or architectural claims into
  empirical proof;
- make a superseded release current;
- grant permissions beyond the copyright and license notices contained in each
  release set;
- imply multi-steward ratification of a substantially single-source canon.

Only a release explicitly identified as current by the active signed canon may be
presented as the current canonical release.
