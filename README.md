`prometheus-canonicals`. This public registry intentionally contains only
release sets selected through the active canonical release procedure.

Historical material is not deleted, rewritten, or silently promoted.

## Claim boundary

A signature establishes signer identity and file integrity. It does not establish
empirical, agronomic, legal, market, financial, security, production, or
multi-steward constitutional validation.

The lowest unresolved gate controls the permitted claim.
