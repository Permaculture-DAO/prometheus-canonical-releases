# Syntropic Sovereignty Header

## Status

This header is retained as a constitutional orientation document for the Prometheus canon and is aligned to the v1.1 White Paper.

## Core statement

Syntropic sovereignty means participation without capture, regeneration without extraction, verification without domination and coordination without reducing living systems or human beings to a single score.

## Canonical boundaries

- Human dignity is greater than any score, signal, token record, wallet history, governance participation, reputation trace or AI classification.
- Ecological evidence governs regenerative value.
- Governance participation does not create financial entitlement.
- PRU does not create land ownership or token ownership.
- TRBK does not create investor exposure unless separately and lawfully structured.
- HoloFuel is an operational infrastructure layer.
